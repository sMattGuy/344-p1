To compile this project run
javac src/*.java -d bin

To run this project from root directory
java -cp bin VoterDriver

VoterDriver is the main thread for this java program
VoterDriver is hard coded with the default values for this project

NAME IN SPECS     NAME IN PROGRAM   VALUE
num_voters 	= numVoters 	  = 20
num_ID_checkers = numIDHelpers 	  = 3
num_k		= numKiosks 	  = 3
num_sm 		= numScanMachines = 4
num_sh 		= numScanHelpers  = 2
